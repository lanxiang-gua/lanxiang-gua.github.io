# 🩺 结核后仍盗汗 · 临床思路口诀

**口诀：**  
👉 **“先复查、辨性质，警复燃、查并病；药副作，内分泌；恶性瘤，若排尽；多为虚，调生活。”**

---

## 🔍 展开解释

1. **先复查、辨性质**
    
    - 胸片/CT、血沉/CRP、痰菌学
        
    - 看盗汗伴随症状：有无低热、咳嗽、体重下降
        
2. **警复燃、查并病**
    
    - 复燃结核？隐匿灶？
        
    - 慢性感染：真菌、HIV、心内膜炎
        
3. **药副作，内分泌**
    
    - 抗结核药物副作用（利福平、异烟肼）
        
    - 甲亢、糖尿病低血糖
        
4. **恶性瘤，若排尽**
    
    - 淋巴瘤、血液系统肿瘤
        
    - 体重下降 + 夜汗 + 发热 = “B症状”
        
5. **多为虚，调生活**
    
    - 植物神经紊乱/康复期体虚
        
    - 睡眠、运动、营养、心理调适
        
    - 中医：心脾两虚 → 归脾丸；阴虚火旺 → 知柏地黄丸
        

---

✅ **记忆要点**：

- **先排病灶，再看并病；排除危重，方归康复。**
    
- [临床上大多数结核治愈后盗汗，最终归因于 **植物神经功能失调+体虚**](文献支持)
